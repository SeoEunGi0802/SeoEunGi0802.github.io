  • 서울시 시장 실증사업

    소속 : 콜로세움코퍼레이션
    기간 : 2022.08 ~ 2022.12
    내용 : "전통시장의 고질적인 배송문제를 해소해 소비자 편의 증진과 시장 활성화를 도모하고자 소비자 요구에 맞춰 당일 및 새벽 배송, 묶음 배송 등 다양한 배송이 가능토록소상인들이 밀집한 전통시장을 중심으로 공동물류를 구현하는 사업"으로 서울시 주관하에 진행한 웹 기반 WMS 개발 프로젝트
    언어 : java 17, Spring Boot, AWS RDS Mysql

    1. 스마트택배의 배송조회 API를 배송추적 API로 교체로 배송 단계 업데이트의 자동화와 일일제한 20번이라는 제약에서 벗어남
    2. JPA도입으로 DB Column 암호화 이슈를 ColumnTransformer어노테이션으로 사용으로 해결 후 문서화 후 공유
    3. N+1이슈 EntityGraph 어노테이션으로 해결 후 문서화 후 공유
    4. AWS EC2 및 S3를 세팅
    5. 주문과 관련된 DB 테이블을 설계
    6. 송장 출력을 위한 규격 임포팅

  • CTK COSMETICS(CTK)

    소속 : 콜로세움코퍼레이션
    기간 : 2022.06 ~ 2024.08
    내용 : 해외 창고에서 COLO 1.0을 이용하려고 하였지만 창고 관리자와 작업자의 요구사항과 실제 업무 진행 단계와 상이하여 새로이 해외 창고 업무에 초점을 두어 개발한 WMS
    언어 : java 17, Spring Boot, AWS RDS Mysql

    1. Zebra 프린트기 SDK 세팅, ZPL을 규격 개발
    2. 상품 관련 정보 테이블이 추가에 따른 종속성 변환으로 상품 체계 변경 작업 진행
    3. 시스템에서 이루어진 재고의 변동 내역 상세를 추출하여 저장하는 MySQL 프로시저 개발
    4. 정산에 필요한 보고서 생성요청 및 다운로드 기능으로 관련 CS건 최소화(대략 90% 감소)
    5. AWS Lambda를 이용한 엑셀 업로드 기능 속도 향상 튜닝으로 리소스 비용 10% 감소 기여
    6. AWS Lambda를 이용한 Slow Query 알람 기능을 개발하여 서비스의 품질향상 도모
    7. Slow Query 알림을 통해 확인된 문제 쿼리를 튜닝하여 최소 10% 최대 97% 속도 향상
    8. RDS CPU, ACU 임계치 초과 알림 기능 개발로 서비스 품질향상 도모
    9. 모니터링 툴 도입 전 logback 라이브러리의 RollingFileAppender를 커스텀하여 로그파일을 rollover 시점에 S3에 저장 하도록 적용 및 S3 버킷에 Life Cycle을 적용
